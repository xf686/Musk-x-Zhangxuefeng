# AI 时代下的高考志愿选择

> 蒸馏了马斯克和张雪峰的高考选志愿skill。选高考志愿用——一个懂时代,一个懂高考;一个负责格局,一个负责落地。

选志愿最痛的,从来不是不知道选啥,是两个声音在你脑子里打架——

一个说:追风口,赌未来,别浪费这个时代;
一个说:先活下来,普通家庭输不起,稳就业要紧。

这个 skill 干脆把这两个声音,变成两位高手给你**会诊**:

- **马斯克(懂时代 · 格局)**——往上看,帮你赌对时代方向。
- **张雪峰(懂高考 · 落地)**——往下拽,帮你以这个分数真能上、真能吃饭。

一个管"该往哪飞",一个管"这架飞机你开不开得起来"。两人各自问诊、各自诊断,最后合议出一份"格局 × 落地"的报告——但拍板权永远在你。

---

## 它和市面上的张雪峰 skill 有什么不同

GitHub 上张雪峰 skill 已经一抓一把,但它们都是**一个人说了算**。本项目的不同,是**交叉视角**:

| | 常见张雪峰 skill | 本 skill |
|---|---|---|
| 人格 | 单人 | 双人格(马斯克 × 张雪峰) |
| 视角 | 就业落地单一视角 | 格局 × 落地 交叉 |
| 时代判断 | 无 | 权威行研事实底座(含乐观、谨慎、唱反调三派) |
| 输出 | 一人建议 | 会诊报告:共识 + 分歧 + 每个方向标"格局分/落地分" |

护城河不在"张雪峰",而在交叉视角:经验(张雪峰)× 数据(麦可思、教育部)× 未来(马斯克 + 权威行研)三方对话,还会如实呈现两人的分歧,而不是喂你一个标准答案。

## 怎么用

把本目录作为 Claude skill 安装后,直接对 Claude 说你的困惑即可,例如:

- "我高考完了不知道选什么专业"
- "计算机和金融,选哪个更有前途?"
- "我想学考古,但家里反对,怎么办?"

Claude 会自动启动四幕会诊:张雪峰问诊 → 马斯克问诊 → 各自诊断 → 合议报告。

---

## 边界(请务必知道)

- 它给的是方向判断和决策框架。
- 它不查、不编任何具体分数线、位次、录取概率——这些请去阳光高考(gaokao.chsi.com.cn)或各省考试院官网核实。
- 它不替你做最终决定,只帮你想清楚。
- 不替代专业的志愿填报服务,仅作思路参考。
