# 《AI 时代下的高考志愿选择》Skill 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development(推荐)或 superpowers:executing-plans 逐任务实现。每个任务用 `- [ ]` 复选框追踪。
> **作者注:** 本项目是 Claude skill(markdown 思维工具),非可执行代码。"测试"= 验收场景:跑一遍模拟会诊,核对输出是否命中设计要求。

**Goal:** 造一个双人格(马斯克 × 张雪峰)会诊式高考志愿 skill,格局 × 落地交叉给建议,纯判断不碰硬数据。

**Architecture:** 一个 `SKILL.md` 调度中枢 + 两个 `personas/*.md` 灵魂文件 + 两个 `data/*.md` 事实弹药文件。三层分离、各自单一职责。会诊走"张雪峰问诊→马斯克问诊→各自诊断→合议报告"四幕。

**Tech Stack:** 纯 Markdown。Claude skill 规范(frontmatter:name + description + 触发条件)。实现时用 `superpowers:writing-skills` 规范写 SKILL.md。读取 GitHub 原始素材用 WebFetch(只读,不 clone、不下载)。

## Global Constraints(每个任务都隐含遵守)

- **不编硬数据**:分数线/位次/录取概率/某校就业率一律不造,输出处写"→ 去[阳光高考/省考试院]核实"。
- **不逐字搬原话**:张雪峰、马斯克的内容**全部蒸馏成方法论,用我们自己的话复述**(公开发布版权红线)。原素材仅作提炼依据。
- **行研数据必须带出处**:③层趋势数据引用时标来源(如"据 WEF《Future of Jobs 2025》")。
- **马斯克严格边界**:绝不碰中国高考具体分数线/录取规则,一涉及即交棒张雪峰。
- **不替用户拍板**:只给方向与利弊,最终决定权留给用户。
- **涉人措辞中性**:对张雪峰本人保持尊重,不渲染未经证实的传闻。
- **语言**:全中文,面向无技术背景用户,多用类比和对比。
